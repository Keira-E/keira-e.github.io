Project Overview

Using the C programming language, I developed an emulator for a rudimentary 16-bit computer system from the ground up. The emulator successfully reads s-record files, converts them into machine code, and subsequently executes the instructions using emulated memory and registers. A key part of the project was implementing custom fault and interrupt handling, which greatly contributed to my fundamental understanding of computer architecture and low-level system operations.
