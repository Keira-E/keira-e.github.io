Project Overview

In a collaborative team setting, I co-designed a device that detects when a user is sitting with improper posture and provides an alert to prompt correction. My primary contribution was inventing and implementing a unique analog switch system. This system used foam-embedded Hall-effect sensors to discreetly detect pressure distribution from where the user was sitting and send detailed feedback to a microcontroller for analysis.

Key Features & Technologies

  • Analog Circuit Design
  • Hall-Effect Sensors
  • Microcontroller Integration
  • System Prototyping
  • Collaborative Design
  • Ergonomic Sensing
