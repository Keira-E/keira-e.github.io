Project Overview

This custom power controller is the primary electronics platform for an underwater computer vision system that monitors fish behavior. It receives high-voltage DC over long cable runs, generates regulated rails for an edge AI compute module, and drives high-power illumination LEDs with tightly controlled timing and current profiles.

I owned the end-to-end design: requirements capture, power architecture, schematics and layout in KiCad, firmware for the STM32 microcontroller, and structured verification including EMI and thermal characterization.

Power Architecture & LED Control

To meet both power and signal integrity constraints over 100 m of cabling, the system uses a high-voltage input rail feeding several cascaded DC-DC stages:

  • High-voltage input rail → intermediate 12 V rail for compute and auxiliary electronics.
  • Dedicated LED rails driving constant-current LED driver ICs for high-intensity illumination.
  • Low-voltage logic rail for the STM32 and digital interfaces.

The STM32 supervises system power states, monitors fault conditions, and generates LED control signals via a parallel SPI interface to multiple LED driver channels. This enables precise control of brightness and timing in synchronization with the AI compute pipeline.

Robustness, EMI, and Field Diagnostics

The PCB layout and component selection were guided by EMI, thermal, and maintainability constraints. Key decisions include:

  • Segregated high-current and low-noise regions with controlled return paths.
  • Use of solid-state switching elements and e-fusing for protection and reliability.
  • Careful placement of bulk and local decoupling to handle long cable transients.

On-board diagnostic LEDs and test points were added to support rapid field troubleshooting, allowing technicians to quickly determine whether issues originate in power distribution, LED channels, or communications.

Key Features & Technologies

  • Multi-rail power architecture for compute, control, and high-power LED driving.
  • STM32-based supervisory control and LED timing management.
  • EMI-aware PCB layout for long cable runs in industrial environments.
  • Integrated protections: e-fuses, solid-state switches, and structured fault handling.
  • Deployed as part of a commercial subsea edge AI system.
