Project Overview

This project focused on developing a wearable sensor system to support an AI-based sleep apnea prediction model. I integrated SpO2 (blood oxygen saturation) and MEMS accelerometer sensors with a PSoC platform. The core of the project was developing robust firmware for real-time signal acquisition, filtering, and processing, ensuring clean and synchronized data streams were available for the predictive algorithm.

Key Features & Technologies

  • Biomedical Sensor Integration (SpO2, MEMS)
  • PSoC Microcontroller Platform
  • Real-Time Firmware Development
  • Digital Signal Processing (DSP)
  • AI/ML Model Support
  • Wearable Device Prototyping
